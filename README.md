# tsVite-tailwind-starter

## Includes

- Typescript
- TailwindCss
- Vite Web Bundler

## Installation

- clone repo

```sh
git clone git@github.com:elameendaiyabu/tsVite-tailwind-starter.git
```

- CD into project

```sh
cd tsVite-tailwind-starter
```

- install dependencies

```sh
npm install
```
